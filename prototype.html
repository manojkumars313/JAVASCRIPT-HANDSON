1)Create 2 objects - parent and child. Attach methods to parent and use those methods in child object using parents prototype 

const parent = {
        detail() {
          console.log(
            `My name is ${this.name} and i am from ${this.location} and my age is ${this.age}`
          );
        },
      };
      const firstChild = Object.create(parent);
      firstChild.name = "EA19";

      firstChild.age = "23";
