3."Write promise function to print numbers from 1 to 7, in which 1 should be printed after 1 sec , 2 should be printed after 2 sec, 3 should be printed after 3 sec and so on. Numbers 1 2 3 4 5 6 7" 

function printNumbersPromise(i) {
                return new Promise((resolve) => {
                    setTimeout(() => {
                        console.log(i);
                        resolve(i);
                    }, i * 1000);
                });
            }
            
            let promise = Promise.resolve();
            
            for (let i = 1; i <= 7; i++) {
                promise = promise.then(() => printNumbersPromise(i));
            }
        
        output:1 2 3 4 5 6 7

4.Create a promise function accepting a argument, if yes is passed to the function then it should go to resolved state and print Promise Resolved, and if nothing is passed then it should go to reject state and catch the error and print Promise Rejected 

const OddEven = new Promise((resolve, reject) => {
            let num = 10;
          
            if (num % 2 === 0) {
              resolve("YES,Promise Resolved");
            } else {
              reject("NO, Promise Rejected");
            }
          });
          
          OddEven.then((result) => console.log(result)).catch((err) => console.log(err));
        
        output: Promise Resolved

5.Create examples to explain callback function

A callback function is a function passed as an argument to another function. It is then called inside the outer function, usually as a way to handle the results of an asynchronous operation
            Exmaple:
            function add(a, b, callback) {
                let result = a + b;
                callback(result);
            }
            
            add(2, 3, function(result) {
                console.log(result); // 5
            });

6.Create examples to explain callback hell function

Callback hell, also known as "Pyramid of Doom," refers to a pattern of deeply nested callback functions, which can make the code difficult to understand, maintain and debug. 
    Example:
    function printNumbersCallbackHell(i, callback) {
        setTimeout(() => {
            console.log(i);
            if (i < 7) {
                callback(i + 1);
            }
        }, i * 1000);
    }
    
    printNumbersCallbackHell(1, function(i) {
        printNumbersCallbackHell(i, function(i) {
            printNumbersCallbackHell(i, function(i) {
                printNumbersCallbackHell(i, function(i) {
                    printNumbersCallbackHell(i, function(i) {
                        printNumbersCallbackHell(i, function(i) {
                            printNumbersCallbackHell(i, function(i) {
                                printNumbersCallbackHell(i, function(i) {});
                            });

                });
            });
        });
    });

7.Create examples to explain promises function

Promises are a modern way of handling asynchronous code in JavaScript. A promise represents the eventual completion (or failure) of an asynchronous operation, and its resulting value. 
    Example:
    const promise = new Promise((resolve, reject) => {
        setTimeout(() => {
            resolve("Hello World");
        }, 1000);
    });
    
    promise.then(result => {
        console.log(result);
    });

8.Create examples to explain async await function

async/await is a more recent addition to JavaScript that makes handling asynchronous code even more convenient, by allowing you to write asynchronous code that looks like synchronous code.
    Example:
    async function sayHello() {
        let message = await delay("Hello World", 1000);
        console.log(message);
    }
    
    function delay(message, time) {
        return new Promise(resolve => {
            setTimeout(() => {
                resolve(message);
            }, time);
        });
    }
    
    sayHello();

    
9.Create examples to explain promise.all function

    
The Promise.all() method is used to handle multiple promises in JavaScript. It takes an array of promises as an argument and returns a single promise that resolves when all the promises in the array have resolved. 
        Example:
        const promise1 = new Promise((resolve, reject) => {
            setTimeout(resolve, 1000, 'Promise 1 resolved');
          });
          const promise2 = new Promise((resolve, reject) => {
            setTimeout(resolve, 2000, 'Promise 2 resolved');
          });
          const promise3 = new Promise((resolve, reject) => {
            setTimeout(resolve, 3000, 'Promise 3 resolved');
          });
          
          Promise.all([promise1, promise2, promise3])
            .then(values => {
              console.log(values); // [ 'Promise 1 resolved', 'Promise 2 resolved', 'Promise 3 resolved' ]
            })
            .catch(error => {
              console.error(error);
            });
